# re-typerooni

A re-implementation of [typerooni](https://github.com/domgetter/typerooni) with [re-frame](https://github.com/Day8/re-frame).

## Requirement

Emacs and Cider installed

## Usage

```bash
$ lein figwheel

$ open http://localhost:3449
```

Emacs

M-x cider-connect
